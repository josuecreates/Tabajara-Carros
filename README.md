# Tabajara Carros - Login

🚗 Sistema de autenticação moderno para o **Tabajara Carros** - Aluguel inteligente.

Interface clean, moderna e responsiva com tema automotivo laranja/preto.

![Preview](https://i.imgur.com/8zK5vL2.png)

## ✨ Funcionalidades

- Design moderno com gradiente e silhueta de carro no fundo
- Animações suaves e efeitos visuais
- Toggle de visibilidade da senha
- Validação em tempo real (front-end)
- "Manter conectado" com LocalStorage
- Recuperação de senha simulada
- Totalmente responsivo (mobile first)
- Acessibilidade básica

## 🛠 Tecnologias Utilizadas

- **HTML5** Semântico
- **CSS3** com variáveis, Flexbox e efeitos avançados
- **JavaScript** (Vanilla)
- Font Awesome (ícones)
- Google Fonts (Inter)

## 🎨 Design

- Paleta laranja temática (#FF6D00)
- Efeito de carro grande semi-transparente no fundo
- Card com glassmorphism leve
- Transições suaves em todos os elementos
